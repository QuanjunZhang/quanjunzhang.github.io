# Quanjun Zhang Homepage

Personal academic homepage for Quanjun Zhang.

## Local Preview

```bash
bundle install
bundle exec jekyll serve
```

Then open:

```text
http://127.0.0.1:4000
```

## Google Scholar Citation Badge

The Google Scholar citation badge is generated by the GitHub Actions workflow in:

```text
.github/workflows/google_scholar_crawler.yaml
```

Set this repository secret before running the workflow:

```text
GOOGLE_SCHOLAR_ID=zRzZ8HwAAAAJ
```
